# 前端交互接口文档 - 里程碑

## 新增里程碑
* cgi/growth/add
* 请求格式
```
{
  'tid': '',                            // 可选；团队ID
  'from': '',                           // 必填；开始时段，精确到天
  'to': '',                             // 可选；结束日期，缺省情况下表示一个时刻
  'title': '',                          // 必填；新闻项，一段文字
  'banner': '',                         // 可选；图片
  'link': '',                           // 可选；外部链接
  'mark': {
    'privacy': '',                      // 可选；状态 0 - 公开，1 - 保护，2 - 私有，缺省表示保护
    'star': ''                          // 可选；加星
  }
}
```
* 响应格式
```
{
  'status': 0,                          // 必填；状态 0 - 成功，1 - 失败
  'message': '',                        // 可选；消息
  'gid': ''                             // 可选；里程碑ID
}
```

## 删除里程碑
* cgi/growth/del
* 请求格式
```
{
  'tid': '',                            // 可选；团队ID
  'gid': ''                             // 必填；里程碑ID
}
```
* 响应格式
```
{
  'status': 0,                          // 必填；状态 0 - 成功，1 - 失败
  'message': ''                         // 可选；消息
}
```

## 标注里程碑
* cgi/growth/mark

OP | 含义
----|----
public | 所有人可见
protected | 导师或密友可见
private | 仅个人可见
star | 加星（event）

* 请求格式
```
{
  'gid': '',                            // 必填；里程碑ID
  'privacy': '',                        // 必填；可见
  'star': ''                            // 可选；加星
}
```
* 响应格式
```
{
  'status': 0,                          // 必填；状态 0 - 成功，1 - 失败
  'message': ''                         // 可选；消息
}
```

## 里程碑详情
* cgi/growth/get
* 请求格式
```
{
  'gid': '',                            // 必填；里程碑 ID
}

```
* 响应格式
```
{
  'status': 0,                          // 必填；状态 0 - 成功，1 - 失败
  'message': '',                        // 可选；消息
  'growth': {
    'title': '',                        // 必填；标题
    'banner': '',                       // 可选；图片
    'summary': '',                      // 可选；摘要
    'author': '',                       // 可选；作者
    'content': '',                      // 必填；正文内容
    'link': '',                         // 可选；外链URL
    'uid': '',                          // 必填；用户ID
    'tid': '',                          // 可选；团队ID
    'mark': {
      'privacy': '',                    // 可选；状态 0 - 公开，1 - 保护，2 - 私有，缺省表示保护
      'star': ''                        // 可选；加星
    }
  }
}
```
