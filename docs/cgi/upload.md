# 前端交互接口文档 - 上传

## 头像上传
* cgi/upload/head
* 请求格式
```
FileUpload Post
```
* 响应格式
```
{
  'status': 0,                          // 必填；状态 0 - 成功，1 - 失败
  'message': '',                        // 可选；消息
  'path': ''                            // 可选；地址
}
```
