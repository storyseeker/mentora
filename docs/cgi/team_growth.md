# 前端交互接口文档 - 团队里程碑

## 关键事件
* cgi/team/growth/event
* 请求格式
```
{
  'tid': '',                            // 可选；团队ID，缺省时表示获取的是Mate（Mate ID是唯一的）
}
```
* 响应格式
```
{
  'status': 0,
  'message': '',
  'event': [
    {
      'from': '',                       // 必填；开始时段，精确到天
      'to': '',                         // 可选；结束日期，缺省情况下表示一个时刻
      'title': '',                      // 必填；新闻项，一段文字
      'link': '',                       // 可选；更多内容链接，可以指向站内或站外
      'mark': {
        'privacy': '',                  // 可选；状态 0 - 公开，1 - 保护，2 - 私有，缺省表示保护
        'star': ''                      // 可选；加星
      }
    }
  ]
}
```

## 成长轨迹
* cgi/team/growth/list
* 请求格式
```
{
  'tid': '',                            // 可选；团队ID，缺省时表示获取的是Mate（Mate ID是唯一的）
  'offset': 0,                          // 可选；偏移量，缺省为0
  'count': 0,                           // 可选；获取条数，缺省时采用服务端配置（如10条等）
  'direct': 0                           // 可选；滑动方向，0 - 更多新事件，1 - 更多旧事件，缺省为1  
}
```
* 响应格式
```
{
  'status': 0,                          // 必填；状态 0 - 成功，1 - 失败
  'message': '',                        // 可选；消息
  'event': [
    {
        'id': '',                       // 必填；编号    
        'from': '',                     // 必填；开始日期
        'to': '',                       // 可选；结束日期
        'title': '',                    // 必填；标题
        'summary': '',                  // 可选；摘要
        'banner': '',                   // 可选；宣传图
        'mark': {
          'privacy': '',                // 可选；状态 0 - 公开，1 - 保护，2 - 私有，缺省表示保护
          'star': ''                    // 可选；加星
        }
    }
  ]
}
```
